HOW TO USE THE PLAYLIST

1. click on the name of the audio track you want to listen to

2. the playlist is an array, when you're adding more tracks, you're adding more values to the array

3. You can stop the track using the 'pause' button

4. you can skip the track using the 'next' button or turn back using the 'previous' button
